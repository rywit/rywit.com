The Statistical Guide to a Silver Buckle

In this paper, we analyze the results of the 2017 Western States 100 Mile Endurance Run. We build a simple classification model which estimates each runner’s likelihood of finishing under 24 hours and earning a coveted silver buckle.

An Analysis of the Western States Lottery

The Western States Endurance Run is America’s most prestegious 100-mile trail ultramarathon. But before runners can conquer 100 miles, they must first conquer the “2^(n-1)” lottery system used to select the entrants. This paper provides a quantitative analysis of several key features of this lottery system.
